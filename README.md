# 🌦️ Weather App

A modern Flutter Weather App that provides real-time weather information for cities around the world. The app allows users to search for locations and view current weather conditions including temperature, humidity, wind speed, and weather descriptions.

## 📱 Features

- Search weather by city name
- Real-time weather updates
- Current temperature display
- Humidity information
- Wind speed details
- Weather condition icons
- Clean and responsive UI
- Error handling for invalid locations

## 🛠️ Technologies Used

- Flutter
- Dart
- Weather API
- HTTP Package
- Material Design
